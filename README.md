# Introducing the threeuk_usage gem

## Usage

    require 'threeuk_usage'

    t = ThreeUKUsage.new
    puts "Three.co.uk Data Allowance:\n\n\tremaining: %s MB\n\texpires: %s" % \
            [t.remaining, t.expires]

Output:

<pre>
Three.co.uk Data Allowance:

        remaining: 11843 MB
        expires: 2018-10-07
</pre>
